About System

This AI-based Missing Person Identification System helps locate missing individuals quickly and efficiently.

🔍 What Problem It Solves

Many missing person cases remain unsolved due to lack of coordination and data. This system centralizes information and helps identify individuals using AI.

⚙️ How It Works

Users can report missing persons, upload images, and search records. The system uses facial recognition to match and identify individuals.

👮 Who Can Use It

Citizens can report and search cases. Police can access advanced tools for investigation and tracking.
